# LazySheet

## Environments

- Production: [https://lazysheet.com](https://lazysheet.com)
- Staging: [https://staging.lazysheet.com](https://staging.lazysheet.com)

## Development

### Prerequisites

- Node.js 18+
- pnpm 8+
- Git

### Setup

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev
```
